# 📊 Tesla & GameStop Stock Analysis

## 📌 Overview

This project analyzes the historical stock prices and revenue of Tesla and GameStop using Python. The data is collected using the yfinance library and web scraping techniques, then visualized to understand trends and patterns.

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* yfinance
* BeautifulSoup
* Requests

## 📈 Features

* Extract stock data using yfinance
* Scrape revenue data from web sources
* Clean and preprocess data
* Visualize stock price and revenue trends

## 📷 Output

The project generates graphs comparing:

* Stock price over time
* Revenue over time

## 🚀 How to Run

1. Install required libraries:

```
pip install yfinance pandas matplotlib beautifulsoup4 requests html5lib
```

2. Run the script or notebook:

```
python historical_Data.py
```

## 📁 Project Files

* `historical_Data.py` – main analysis script
* `README.md` – project documentation

## 🎯 Conclusion

The analysis shows how stock prices and company revenue can behave differently, especially in cases like GameStop where market trends influence stock value.
